■課題4の説明

1. kadai4.shについて
最大公約数を算出するスクリプトです。
以下のように、引数を2つ与えて実行することで機能します。

bash /home/kf32/kadai4/kadai4.sh 100 8
→4が回答される


2. checker.shについて
kadai4.shをテストするスクリプトです。
引数が無かった場合、引数が1つだけしか与えられなかった場合、
0と0を与えた場合でテストを行います。
